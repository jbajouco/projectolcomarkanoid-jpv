int install_asm_irq_handler ( int  irq,
void(*)(void)  irq_func,
_go32_dpmi_seginfo *  old_irq 
)

Installs the ASM written function 'irq_func' as the interrupt handler associated with interrupt 'irq', updating the 'old_irq' structure with the current interrupt handler. Returns 0 on success or -1 on error, in which case the original interrupt handler remains installed.

See reinstall_asm_irq_handler() to reinstall the original handler.
